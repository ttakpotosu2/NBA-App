# NBA App - Challanging myself with apis that are "harder" to work with.

#Technology Stack
1. Retrofit
2. Coil
3. RapidApi - API_NBA (https://rapidapi.com/api-sports/api/api-nba)
4. Hilt
5. Jetpack Compose
6. Animated Navigation Bar Library
